# Description

`push_swap` is a sorting algorithm project where the goal is to sort a
list of unique integers using two stacks (Stack A and Stack B) and a
restricted set of stack operations. The program must output a sequence
of valid instructions that sorts the numbers in ascending order in Stack
A while leaving Stack B empty.

The challenge is not only to sort the numbers correctly but to do so
using as few operations as possible.
